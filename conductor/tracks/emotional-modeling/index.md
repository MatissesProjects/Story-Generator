# Track: Emotional Modeling (OCC Framework)

## Objective
Implement the **Ortony, Clore, and Collins (OCC)** emotional model to provide characters with authentic interiority and emotional depth that directly influences the generated narrative.

## Key Goals
- [ ] Define **Emotional Appraisal Rules** based on story events (e.g., Goal Congruence, Agency).
- [ ] Implement an **Emotional State Machine** for each major NPC.
- [ ] Update LLM prompts to include character affective states (e.g., "conflicted but polite").
