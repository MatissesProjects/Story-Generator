# Track: HTN Planning (Hierarchical Task Networks)

## Objective
Transition from linear milestones to a dynamic, hierarchical planning system that allows the Director to maintain narrative coherence while adapting to complex player interactions.

## Key Goals
- [x] Implement a **Domain Definition** for narrative goals (e.g., Heist, Investigation).
- [x] Build a **Hierarchical Planner** that decomposes high-level goals into primitive tasks.
- [x] Integrate with `agency_engine.py` (via Director) to coordinate character actions toward larger plot goals.
