# Track: HTN Planning (Hierarchical Task Networks)

## Objective
Transition from linear milestones to a dynamic, hierarchical planning system that allows the Director to maintain narrative coherence while adapting to complex player interactions.

## Key Goals
- [ ] Implement a **Domain Definition** for narrative goals (e.g., Heist, Investigation).
- [ ] Build a **Hierarchical Planner** that decomposes high-level goals into primitive tasks.
- [ ] Integrate with `agency_engine.py` to coordinate character actions toward larger plot goals.
