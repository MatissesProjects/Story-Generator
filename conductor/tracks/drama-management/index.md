# Track: Drama Management (Tension Curve)

## Objective
Implement a "Drama Manager" that explicitly monitors and manipulates the **Narrative Tension Curve** to ensure a balanced and engaging experience.

## Key Goals
- [ ] Build a **Tension Monitor** that evaluates turns for action vs. meaning beats.
- [ ] Implement a **Pacing Controller** to adjust LLM constraints (e.g., shorter vs. longer outputs).
- [ ] Prevent "Action Fatigue" by triggering mandatory "Lull Beats" (character building/rest).
