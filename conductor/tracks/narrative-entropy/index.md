# ⏳ Track: Narrative Entropy

## Overview
Introduce dynamic decay and conflict resolution to the story's memory and lore, ensuring the world is not static.

## Core Objectives
- **Relationship Decay**: Scores drift back to neutral over time if not maintained.
- **Lore Volatility**: Minor facts can be forgotten or distorted.
- **Lore Duels**: Automated resolution of contradictions between generation and history.
